## 通用结论

大厂 C++ 面试里，算法最常出现的不是特别偏的冷题，而是这些主线：
- 哈希
- 堆 / TopK
- 树 / 红黑树 / 遍历
- BFS / DFS
- DP
- 字符串与数组中等题
- 手写线程池 / LRU / 生产者消费者这类实现题

所以如果时间有限，优先把：
1. 哈希 + 堆
2. 树 + 图搜索
3. DP
4. 手撕实现题

这四块打透，收益最高。
