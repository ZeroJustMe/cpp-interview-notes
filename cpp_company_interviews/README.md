# 中大厂 C++ 面经真题整理

这份目录是从公开技术社区里二次整理出来的**公司维度面经速查表**，目标不是替代完整八股，而是把：

- 公司分类
- 公开面经里反复出现的真题
- 每题 1～2 句速答
- 对应仓库内已整理的系统知识链接

放到一处，方便临近面试时按公司快速过一遍。

## 目录结构

- `companies/`：按公司分类整理的面经真题
- `高频知识点.md`：跨公司反复出现的高频主题汇总
- `算法题映射.md`：面经题型到 LeetCode/刷题方向的映射
- `sources.md`：本轮查到并实际参考的公开社区来源

## 使用方式

### 1. 临近某家公司面试
先看：

- `companies/<公司>.md`

### 2. 想抓大厂共性高频题
再看：

- `高频知识点.md`

### 3. 想展开复习某个知识点
跳回主仓库：

- `../cpp_interview_notes/`

## 说明

1. **结构化真题主要来自牛客公开面经帖**，因为牛客的公开面经标题和摘要更适合做公司维度检索。
2. **知乎、力扣这轮主要作为补充社区入口和热点交叉验证**；它们更适合补充岗位认知和算法题单，不如牛客容易直接抽取结构化面经题目。
3. 每个问题下的回答都尽量压到 1～2 句话，定位是**面前速刷**；真正展开复习请点到主仓库内的详细笔记。

## 推荐阅读顺序

1. 先看 `高频知识点.md`
2. 再看目标公司的 `companies/*.md`
3. 顺手看 `算法题映射.md`，把面经题型映射到刷题方向
4. 最后回主仓库的专题文件补细节

如果后面继续打磨，这个目录还可以继续补：

- 按岗位细分（客户端 / 后端 / 基础设施）
- 每家公司按一面 / 二面 / HR 面区分
- 真题对应手写题与算法题单
