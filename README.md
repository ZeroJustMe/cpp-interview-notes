# cpp-interview-notes

这是一个面向 C++ / 后端 / 系统方向面试准备的知识仓库。

## 目录

- `cpp_interview_notes/`：主知识库，按语言、操作系统、网络、数据库、架构、算法、岗位路径分专题整理
- `cpp_company_interviews/`：按公司整理的公开面经真题速查表，以及高频知识点、算法题映射、来源清单

## 使用建议

1. 先看 `cpp_interview_notes/README.md` 抓主线
2. 再按目标公司看 `cpp_company_interviews/companies/*.md`
3. 面前速刷时优先看 `cpp_company_interviews/高频知识点.md`

## 当前定位

这不是单纯的八股关键词清单，而是尽量把：
- 高频问题
- 追问链
- 工程语境
- 复习路径

整理成一套更适合真正面试准备的资料。
