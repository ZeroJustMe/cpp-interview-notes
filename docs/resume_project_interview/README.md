## 覆盖的项目技术栈

- **向量流并行连接引擎**：滑动窗口、双层并发索引、SPSC 无锁队列、工厂模式插件化、多节点扩展
- **AI 推理编排框架**：PyBind11 跨语言封装、CMake 构建、Pipeline 服务化
- **CPU-GPU 协同 ANNS**：CUDA 编程、异构计算任务划分、向量索引
- **自动驾驶仿真器（L4/AVP/L2）**：MVC 架构、ROS/Cyber 中间件、Protobuf、Failure Detector、IDM 交通流模型、仿真一致性

## 使用建议

1. **模拟面试**：找搭档按追问链逐层深入，练习当场组织答案的能力
2. **自测深度**：每道追问都尝试回答，答不上来的点点击 📎 链接跳转到详细笔记补上
3. **串联复习**：项目深挖题 → 八股题 → 场景题，三者互相引用，形成闭环
4. **项目结合**：八股文最后一节"项目结合扩展"专门训练"把八股和项目连起来讲"的能力
