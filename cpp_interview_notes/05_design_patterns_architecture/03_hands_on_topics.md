# 03. 手撕题与实现专题导航

## 1. 手撕线程池会考什么？

- 任务队列
- worker 线程循环
- 条件变量等待/唤醒
- 停止标志
- 析构时优雅退出
- 防止任务丢失

## 2. 手撕 shared_ptr 会考什么？

- 控制块
- 引用计数
- 拷贝构造 / 赋值
- 析构时归零释放
- weak_ptr 扩展思路

## 3. 手撕 LRU 会考什么？

### 标准结构
- 哈希表 + 双向链表

### 复杂度
- 查询 O(1)
- 插入 O(1)
- 淘汰 O(1)

## 4. 手撕单例会考什么？

- 懒加载
- 线程安全
- 拷贝禁用
- 生命周期

## 5. 手撕生产者消费者会考什么？

- 共享队列
- 互斥锁
- 条件变量
- 虚假唤醒
- 退出协议

## 6. 面试提示

真正手撕时，正确性优先于“炫技”。先写出：
- 数据结构
- 并发边界
- 生命周期控制
- 异常/退出路径
